# Skripsi-2026
# Sentiment Analysis of Indonesian Users Toward Five Artificial Intelligence Applications Using Machine Learning and IndoBERT

This repository contains the implementation of an undergraduate thesis on sentiment analysis of Indonesian user reviews for five AI applications (ChatGPT, Google Gemini, Grok, Microsoft Copilot, and Meta AI).

## Features
- Google Play Store review scraping
- Text preprocessing
- TF-IDF feature extraction
- Sentiment classification using Naive Bayes, SVM, and IndoBERT
- Aspect and use case analysis
- Streamlit dashboard for visualization

## Technologies
- Python
- Scikit-learn
- Transformers (IndoBERT)
- Pandas
- NLTK
- Sastrawi
- Streamlit

## Results
| Model | Accuracy |
|-------|---------:|
| IndoBERT | **90.30%** |
| Naive Bayes | 88.84% |
| SVM | 80.62% |
